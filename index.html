Introduction

Web Traffic Signs (WTra) is a web project, part of a Web Application Development course studied by master students enrolled at Faculty of Computer Science .
The project is about developing a (micro)service-based Web system where user(s) can introduce a video recording where is captured an urban route. The input video will be processed and the traffic signs will be recognized and reported. For detection will be used a specific public API. With the information that is gathering, a specific ontology specified in OWL will be created. For each recognized road sign, a SPARQL endpoint will offer various information about it like meaning, type, legal regulations, context of use etc.
